# 🥫 Vorratsverwaltung – Home Assistant Add-on

Vollständige Haushalts-Vorratsverwaltung mit Rezepten, Einkaufslisten und Web-Import.

## ✨ Features

- 📦 **Vorratsverwaltung** – Produkte mit Menge, MHD-Warnungen, Mindestmengen
- 🛒 **Einkaufslisten** – Mehrere Listen, Preise, Drag & Drop, automatisch in Vorrat buchen
- 📖 **Rezepte** – Vorrat-Abgleich, PDF-Import, Web-Import (Chefkoch, Kaufland, Lidl, u.v.m.)
- 🌍 **Mehrsprachig** – Deutsch, Englisch, Französisch, Spanisch
- 💶 **Mehrere Währungen** – €, CHF, £, $
- 📱 **Mobil-optimiert** – Bottom-Navigation, als PWA installierbar

## 🔧 Installation

1. In Home Assistant: **Einstellungen → Add-ons → Add-on Store → ⋮ Menü → Repositories**
2. URL hinzufügen: `https://github.com/jenser1/ha-vorrat-addon`
3. **Vorratsverwaltung** installieren & starten
4. Über die HA-Sidebar öffnen

## 📱 Als App installieren

Im Browser auf **"Zum Startbildschirm hinzufügen"** tippen – öffnet dann wie eine native App.

## 🔄 Updates

Updates erscheinen automatisch im HA Add-on Store wenn eine neue Version veröffentlicht wird.
